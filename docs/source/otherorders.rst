Other Orders
============
The **Other Orders** feature allows the admin to:

.. toctree::
   :maxdepth: 3

1. **Upload** - upload other order files from a hospital or client EHR system that can be processed to provide valid and invalid procedures.

.. image:: otherorderupload.png 
   :width: 500px
   :align: center
   :height: 200px

2. **Contents** - view Valid,invalid and On Hold procedure data and send to the provider for review.

.. image:: otherordercontent.png    
   :width: 500 px
   :align: center
   :height: 200px

3. **Review** - list files that have been assigned for review and their status.

.. image:: otherorderreview.png
   :width: 500px
   :align: center
   :height: 200px
